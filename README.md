# KaseiCoin

The objective of this excercise is to use blockchain technology to define a new cryptocurrency named KaseiCoin. KaseCoin is created as fungible token thatâ€™s ERC-20 compliant. It launches a crowdsale that will allow people to convert their money to KaseiCoin.

It is important to note I did not use MetaMask as we did not cover it in class.

## Libraries and Dependencies 
1) Solidity
2) Remix IDE
3) Ganache

## Steps 
1) Create the KaseiCoin Token Contract
2) Create the KaseiCoin Crowdsale Contract
3) Create the KaseiCoin Deployer Contract
4) Deploy and Test the Crowdsale on a Local Blockchain
